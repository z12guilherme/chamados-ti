# Manual do Usuário - Sistema de Chamados de T.I.

Bem-vindo ao nosso sistema de chamados! Este guia rápido irá ajudá-lo a usar todas as funcionalidades da plataforma, seja você um usuário abrindo um chamado ou um administrador gerenciando as solicitações.

---

## Para Usuários (Abrindo e Consultando um Chamado)

### 1. Como Abrir um Novo Chamado

Qualquer pessoa pode solicitar suporte de T.I. de forma rápida e fácil.

1.  **Acesse a Página Inicial**: Abra a página principal do sistema.
2.  **Preencha seus Dados**: Informe seu nome completo e o setor onde você trabalha.
3.  **Descreva o Problema**: No campo "Descrição do Problema", seja o mais detalhado possível. Informe qual equipamento apresenta defeito, o que estava fazendo quando o erro ocorreu e qualquer mensagem que tenha aparecido na tela.
4.  **Defina a Urgência**: Selecione o nível de urgência que você considera para o seu problema:
    *   **Baixa**: O problema não impede suas tarefas diárias.
    *   **Média**: O problema atrapalha, mas você ainda consegue trabalhar parcialmente.
    *   **Alta**: O problema impede totalmente que você realize seu trabalho.
5.  **Envie o Chamado**: Clique no botão "Abrir Chamado".

Após o envio, uma mensagem de sucesso aparecerá na tela com o **número do seu protocolo**. **Guarde este número!** Ele é essencial para consultar o status do seu atendimento.

### 2. Como Consultar o Status de um Chamado

1.  **Acesse a Página de Consulta**: No rodapé da página, clique em "Consultar um chamado" ou acesse diretamente a página de consulta.
2.  **Informe o Protocolo**: Digite o número do protocolo que você recebeu ao abrir o chamado (ex: `CH-2025-ABCDEF`).
3.  **Consulte**: Clique em "Consultar". O sistema exibirá os detalhes e o status atual do seu chamado.

## Para Administradores (Gerenciando os Chamados)

### 1. Acessando o Painel

O painel é a central de gerenciamento de todos os chamados. Para acessá-lo, você precisa de um usuário e senha fornecidos pela equipe de T.I.

1.  Acesse a página de **Login** (geralmente `login.html`).
2.  Insira seu email e senha de administrador.
3.  Você será redirecionado para o **Painel Administrativo**.

### 2. Gerenciando os Chamados

No painel, você verá os chamados organizados por status. Para cada chamado, você pode:

*   **Alterar o Status**: Use os botões no card do chamado para mudar seu estado (ex: de "Pendente" para "Em Andamento").
*   **Resolver um Chamado**: Ao clicar em "Resolver", uma janela aparecerá para que você descreva a solução aplicada. O chamado será movido para a seção "Concluídos".
*   **Aguardar Peça**: Se a solução depende de uma peça, use esta opção para registrar qual item está pendente. O chamado ficará no status "Aguardando Peça".
*   **Excluir um Chamado**: Use o botão de lixeira para remover um chamado permanentemente (use com cuidado!).
